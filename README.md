## :jigsaw: ¿Qué es un programa?

Un **programa** es una secuencia de instrucciones que le indica a la computadora qué hacer paso a paso.  
Cada programa tiene un propósito, como calcular, procesar datos o mostrar información.  
Por ejemplo, cuando hacemos clic en un botón, detrás de ese evento hay un código que le dice al sistema qué acción ejecutar.

Los programas se basan en **algoritmos**, que son como recetas que detallan las acciones que deben realizarse en orden lógico.

---

## :brain: Variables en programación

Las **variables** son espacios en memoria donde se guardan datos que pueden cambiar durante la ejecución del programa.  
Por ejemplo, una variable puede almacenar un nombre, un número o un resultado matemático.

En Python, las variables no requieren que se indique su tipo (como en otros lenguajes), lo que las hace muy flexibles.  
Aquí tienes un ejemplo:


```python
# Ejemplo simple de variables y concatenación en Python
nombre = "Kate"
edad = 22
mensaje = "Hola, mi nombre es {nombre} y tengo {edad} años."
print(mensaje)

:small_blue_diamond: **Explicación:**

- `nombre` y `edad` son variables.  
- El texto dentro de las llaves `{}` se sustituye por el valor real.  
- `print()` muestra el mensaje en la pantalla.
```
